## About Me

Frontend Developer especializado en **Angular** y **Next.js**, con capacidad para construir soluciones Full Stack usando **Java** y **Spring Boot**. Aprendiz del programa **Tecnólogo en Análisis y Desarrollo de Software (ADSO)** en SENA Armenia, Quindío 🇨🇴.

Me enfoco en construir interfaces modernas, accesibles y bien estructuradas — y cuando el proyecto lo requiere, también diseño y conecto el backend.
